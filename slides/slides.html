Hackety ♥ Buttons

Saucy buttons

button "Don't click"

Ok, so do something

button "Don't click" do
  alert "Hey! No fair!"
end

Be the button

b = button "Don't click" do
  alert "Fine. Bye."
  b.remove
end

Play hard to get

b = button ":P" do
  left = rand(width)
  top = rand(height)
  b.move left, top
end
